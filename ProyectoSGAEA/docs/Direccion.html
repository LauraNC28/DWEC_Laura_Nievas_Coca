Class: Direccion

Direccion(calle, numero, pisoopt, codigoPostal, provincia, localidad)

La clase `Direccion` representa una dirección física con atributos como calle, número, piso, código postal, provincia y localidad. Incluye validaciones y métodos para acceder y mostrar la información de manera estructurada.
